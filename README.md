# Установка

Скопируйте репозиторий в папку на локальном компьютере. Лучше с помощью git:
```
git clone https://github.com/AltmanEA/course_typescript_lab
```
Перейдите в папку и установите необходимые модули
```
cd course_typescript_lab
npm install
```
Для проверки решения задачи запустите соответствующий тест, например:
```
npm run test basic01
```
Для прохождения контрольных установите в vscode расширение ```examenator-X.X.X.vsix``` (можно просто перетащить этот файл на вкладку "Extentions").

# Рекомендации по решению

Решать задачи нужно с помощью тех средств языка программирования, которые изучаются в текущем блоке. Например, если изучаются циклы, то не стоит использовать рекурсию. Хотя это и не спортивное программирование и оптимизация решений не требуется, тем не менее не стоит приводить решение в лоб. При проведении контрольных работ корректность решения будет выборочно проверятся вручную.

Решать задачи рекомендуется самостоятельно. Смысл заданий - набить руку на решении подобных задач и работе с typescript. Любой ИИ сможет легко решить эти задачи (не всегда корректно в терминах предыдущего абзаца), но просмотр чужого решения не сформирует нужные нейронные связи для естественного интеллекта.
